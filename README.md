# 🎓 Student Management System (MERN)

A full-stack **Student Management System** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**. The application allows users to register, log in, and perform CRUD operations on student records.

# 📋 Prerequisites

Before running the project, install the following software:

### 1. Node.js

Download and install the latest LTS version:

https://nodejs.org/

Verify installation:

```bash
node -v
npm -v
```

---

### 2. MongoDB

Download MongoDB Community Server:

https://www.mongodb.com/try/download/community

(Optional) Download MongoDB Compass:

https://www.mongodb.com/products/compass

Verify installation:

```bash
mongod --version
```

---

# 📥 Clone the Repository

```bash
git clone https://github.com/RkVinayak/Student-Management-Using-MERN.git
cd Student-Management-Using-MERN
```

---

# ⚙️ Install Dependencies

## Backend

```bash
cd backend
npm install
```

This installs:

* Express.js
* Mongoose
* CORS
* Dotenv
* Other backend dependencies

---

## Frontend

Open another terminal:

```bash
cd frontend
npm install
```

This installs:

* React.js
* Axios
* React Router DOM
* Vite
* Other frontend dependencies

---

# ▶️ Running the Project

## Step 1: Start Backend

Open Terminal 1

```bash
cd backend
node server.js
```

Backend will run on:

```text
http://localhost:5000
```

---

## Step 2: Start Frontend

Open Terminal 2

```bash
cd frontend
npm run dev
```

Frontend will run on:

```text
http://localhost:5173
```

---

## Step 3: Open the Application

Open your browser and visit:

```text
http://localhost:5173
```
